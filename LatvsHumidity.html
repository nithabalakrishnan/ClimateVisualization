Summary: Latitude vs.X

The purpose of this project was to analyze how weather changes as you get closer to the equatr. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset over 500 cities.
After assmbling the dataset, used Matplotlib to plot various aspects of the weather vs. lattitude. Factors we looked at included: temperature, cloudiness,wind,and humidity. This site provides the source data and visualizations created as part of the the analysis, as well as explanations and descriptions and any trends and correlations witnessed.
